#Koa kickstart

Quick fiddle is a html/js/css fiddler that includes the following : 
Application template for the following stack
- angular
- bootstrap
- animate.css
- less
- bower
- gulp
- koa


##Usage
- clone the git repo : `git clone git@github.com:hgrubst/quickFiddle.git`
- run `npm install`
- run gulp serve
